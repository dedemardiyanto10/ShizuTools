Privacy Policy for ShizuTools

Terakhir Diperbarui: 6 April 2026

Privasi Anda sangat penting bagi kami. Kebijakan Privasi ini menjelaskan bagaimana ShizuTools mengelola informasi di dalam aplikasi.

1. Pengumpulan Data

ShizuTools adalah aplikasi utilitas luring (offline-first). Kami TIDAK mengumpulkan, menyimpan, atau mengirimkan data pribadi Anda (seperti nama, email, atau lokasi) ke server mana pun.

2. Fitur Keamanan Biometrik

Aplikasi ini menggunakan API Biometrik resmi Android untuk fitur penguncian. Kami tidak memiliki akses, tidak menyimpan, dan tidak dapat melihat data sidik jari atau wajah Anda. Semua proses verifikasi dilakukan sepenuhnya oleh sistem operasi Android.

3. Fitur Pembersihan Cache

Fitur "Clear Cache" hanya menghapus file sementara yang dibuat oleh aplikasi untuk mengosongkan ruang penyimpanan. Fitur ini tidak akan menghapus file pribadi, foto, atau dokumen pengguna di luar direktori cache aplikasi.

4. Izin Aplikasi (Permissions)

  • USE_BIOMETRIC: Digunakan hanya untuk memverifikasi identitas Anda saat membuka fitur yang terkunci.
  • INTERNET: Digunakan terbatas untuk fungsionalitas aplikasi yang memerlukan koneksi (jika ada).

5. Keamanan

Semua preferensi pengaturan Anda disimpan secara lokal di perangkat Anda sendiri. Data ini akan terhapus secara otomatis jika Anda menghapus instalan aplikasi.

6. Kontak

Jika ada pertanyaan mengenai kebijakan ini, Anda dapat menghubungi pengembang melalui email atau halaman GitHub resmi proyek ini.
